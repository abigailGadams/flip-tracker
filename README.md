# FlipTimeline

Project management for real estate flippers — track renovation tasks, budgets, and timelines.

## Features

- **Dashboard** — Overview stats, phase progress bars, and budget tracker
- **Task Management** — Tasks grouped by renovation phase with checkboxes, due dates, and budgets
- **Timeline** — Gantt-style visualization of all tasks across phases
- **Budget Breakdown** — Per-phase cost analysis, budget bar, and top expenses
- **Flip Templates** — Pre-built templates for Standard Flip, Cosmetic Refresh, and Gut Rehab

## Getting Started

```bash
npm install
npm run dev
```

Opens at [http://localhost:3000](http://localhost:3000).

## Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 14 (App Router)
- React 18
- Lucide React (icons)
